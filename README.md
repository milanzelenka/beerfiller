# beerfiller - ESPHome project for Kincony A16 board

Beerfiller is automation of bottle filling machine used in small breweries. It uses Kincony A16 board with some SSR relay for control of a few solenoid valves and contactors.


## Input mediums - controlled by solenoid valves
- Beer
- Innert Gas (CO2)

## Output Solenoids Valves
- Drain (*full output*)
- Drizzle (*reduced output*)
